# MSBD566 Predictive Modeling and Analytics

This repository contains coursework for MSBD566, focusing on data analysis, visualization,
and performance comparisons using Python and Pandas.

## Assignment 1
Assignment 1 explores Nashville Air Quality Index (AQI) and pollen data using Pandas.
The analysis includes data visualization, descriptive statistics, vectorized vs manual
computation comparisons, and interpretation of environmental data trends.
